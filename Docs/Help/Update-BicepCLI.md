# Update-BicepCLI

## SYNOPSIS

Update Bicep CLI (Windows only)

## SYNTAX

```powershell
Update-BicepCLI [<CommonParameters>]
```

## DESCRIPTION

Update-BicepCLI is a command to update to the latest Bicep CLI release available from the Azure/Bicep repo.

## EXAMPLES

### Example 1

```powershell
Update-BicepCLI
```
