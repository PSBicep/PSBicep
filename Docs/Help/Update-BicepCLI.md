# Update-BicepCLI

## SYNOPSIS
Update Bicep CLI

## SYNTAX

```powershell
Update-BicepCLI
```

## DESCRIPTION
Update-BicepCLI is a command to update to the latest Bicep CLI release available from the Azure/Bicep repo.

## EXAMPLES

### Example 1
```powershell
Update-BicepCLI
Update Bicep CLI
```
